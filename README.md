## Cách chạy project

### 1. Cài đặt dependencies
```bash
yarn install
# hoặc
npm install
```

### 2. Tạo file `.env`
```env
DB_HOST=localhost
DB_PORT=3306
DB_NAME=linkoma_db
DB_USER=root
DB_PASSWORD=yourpassword
JWT_SECRET=your_jwt_secret
```

### 3. Chạy project
```bash
yarn dev
# hoặc
npm run dev
```
